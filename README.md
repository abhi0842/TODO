# TODO List API

## Overview
This is a Spring Boot-based REST API for managing a to-do list. It supports basic CRUD operations with MongoDB as the database.

## Features
- Add a new task
- View all tasks
- View task by ID
- Update task status
- Delete a task

## Technologies Used
- Java
- Spring Boot
- MongoDB

## Prerequisites
- Java 17+ installed
- Maven installed
- MongoDB running locally or via Docker
